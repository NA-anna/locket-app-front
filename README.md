# locket-app-front

### 가는날이장날 app
- 당신도 셀러가 될 수 있습니다!
- local + market 
- 오프라인 플리마켓/벼룩시장, 비상설 마켓정보 플랫폼

1. 사용기술
- swift (MVC모델, alamofire)
- 사용 API - open API - 카카오 맵, 카카오 장소검색, 공공데이터 전국5일장

2. 주요기능 
- 마켓 등록 : 플리마켓/벼룩시장 등의 마켓을 등록
- 셀러 모집, 셀러 신청
- 오프라인 마켓 정보를 지도와 목록으로 표시
